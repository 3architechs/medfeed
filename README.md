# MedFeed

> Real findings. No filters. Find a doc who thinks for themselves.

A single-page web aggregator that surfaces health and medical findings from X (Twitter) in the last 24 hours, presented for a general public and patient audience.

---

## Files

| File | Purpose |
|---|---|
| `index.html` | Full markup — sidebar, topbar, feed, archive drawer, footer |
| `styles.css` | All styles — design tokens, layout, cards, archive, dark mode |
| `app.js` | Sidebar toggle + tooltips, search, archive drawer, dark mode, timestamps |

---

## Design system

**Direction:** Warm Airy (Direction C — locked, do not change)

| Token | Value |
|---|---|
| Base background | `#FFF8F4` |
| Primary accent (coral) | `#D85A30` |
| Independent warning red | `#E24B4A` |
| Dark mode base | `#141210` |
| Headline font | Lora (serif, Google Fonts) |
| Body font | Inter (sans, Google Fonts) |

Sidebar: 52px collapsed / 220px expanded  
Archive drawer: cool slate palette (`#F4F6F9` / `#6A88B8`) — intentionally distinct from live peach feed

---

## Key features (all in mockup)

- Collapsible sidebar with specialty, kind, and source filters
- Hover tooltips on collapsed dots (`position:fixed` to escape clip boundary)
- Live search with 280ms debounce, synonym matching, coral highlights
- Dark mode toggle (warm near-black `#141210`)
- Two feed sections: Independent Voices (red left border) + Institutional & Peer-Reviewed
- Card anatomy: specialty pill → title → "What this means for you" coral box → meta row
- Collapsible warning bars on independent cards
- Card expand: full post text, thread pagination, "Open on X" CTA
- Archive drawer (slide-in from right) with 48h/72h/7d/custom range chips
- Footer collapsed by default

## Production build notes

### X scraping
- Scrape window: last 24 hours only
- Content-type badges (Breakthrough / Research study / Clinical trial / Safety update) should be classified by LLM at scrape time
- "What this means for you" boxes should be generated by LLM per post

### Search
- Mockup uses keyword + synonym matching
- Production: replace with embedding-based semantic search

### Accessibility
- WCAG 2.1 AA audit deferred to build phase
- Do not apply accessibility fixes to the mockup — it breaks the visual design

### Icons
- Inline SVGs only — no icon fonts (Tabler/similar fail to render in some environments)

### Storage
- No localStorage used in mockup
- Production can use localStorage for filter state, dark mode preference
